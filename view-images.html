Image gallery

Serve this folder over HTTP (e.g. serve-viewer.ps1), then Scan folders, pick a directory, and Load images. Lists .png .jpg .jpeg .webp .gif .bmp in that folder (not subfolders). In full screen (gallery panel): ←→ change folder and reload images (ignored in text fields); ↑↓ scroll the gallery; Space toggles Actual size (ignored in text fields and on the Actual size checkbox). Esc exits full screen. On touch, tap the top or bottom edge of the image area to open the same controls (auto-hides after a few seconds or Hide). Actual size shows each image at its pixel dimensions in rows that wrap (scroll horizontally if a row is wider than the panel).
